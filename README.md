# React Storybook Demo

A demo repository to explore **React component development** with **Storybook**. Includes examples of reusable components like **Button**, **Card** and **Stack**, with dynamic controls, TypeScript support, and interactive stories to visualize and test components in isolation.

---

## Features

- React + TypeScript components  
- Storybook setup with CSF3 format  
- Controls for dynamic props  
- Examples with children, layouts, and styles  
- Tailwind CSS & custom styling support  

---

## Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/asadrehman1/React-Storybook.git
cd react-storybook
npm install
```

---

## Usage

Run the Storybook development server:

```bash
npm run storybook
